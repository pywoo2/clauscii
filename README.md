# clauscii

Cute ASCII art & animations for [Claude Code](https://docs.anthropic.com/en/docs/claude-code). Brightens every session with a random piece of art on startup.

```
  /\_/\
 ( o.o )    hey, let's build something cool
  > ^ <
```

## Install

```
/plugin marketplace add pywoo2/clauscii
/plugin install clauscii@pywoo2-clauscii
```

## What it does

Every time you start a new Claude Code conversation, you'll be greeted with a random cute ASCII art piece — cats, bunnies, robots, dinosaurs, and more.

You can also ask for art anytime:
- `/art` — show a random art piece
- "show me some ascii art"
- "another one"
