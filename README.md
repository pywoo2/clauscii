# clauscii

ASCII art for [Claude Code](https://docs.anthropic.com/en/docs/claude-code). Random art on startup, every session.

```
~ ngl judging your code rn ~

            .                .
            :"-.          .-";
            |:`.`.__..__.'.';|
            || :-"      "-; ||
            :;              :;
            /  .==.    .==.  \
           :      _.--._      ;
           ; .--.' `--' `.--. :
```

## Install

```
/plugin marketplace add pywoo2/clauscii-marketplace
/plugin install clauscii@clauscii-marketplace
```

## What it does

Every time you start a new Claude Code conversation, you get greeted with a random ASCII art piece. You can also use `/art` anytime.
