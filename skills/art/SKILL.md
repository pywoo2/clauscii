# Clauscii — Show Art

Pick a random piece from the startup-art gallery and display it in a code block with a fun one-liner caption.

If the user asks for something specific (e.g. "show me a cat"), pick the closest match from the gallery. If nothing matches, improvise a cute ASCII art piece in the same style.
