# Clauscii — Show Art

Pick a random art piece from the startup-art gallery.

IMPORTANT: You MUST wrap the art in a fenced code block (triple backticks) to preserve spacing and prevent markdown from eating backslashes. Never output ASCII art as plain text.

If the user asks for something specific (e.g. "show me a cat"), pick the closest match from the startup-art gallery. If nothing matches, improvise something cute in the same style — always in a code block.
